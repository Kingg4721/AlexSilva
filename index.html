Ember Caves

Description

A 2D pixel-art dungeon platformer in Unity. Explore platforms, avoid hazards such as lava and traps, and progress through doors between areas. The project focuses on mechanics (movement, gravity inversion, damage, animation, inventory, and statue interaction), atmosphere, and a foundation to expand with new levels and features.

Video

What I did

  • Mechanics: Movement, traps, moving environment parts, gravity inversion, damage system.
  • Obstacles: Implemented all obstacles (spikes, lava, lava drops, etc.).
  • Tilemaps: Built the tilemaps (art not created by me).
  • Character & animations: Created character animations and death fade in/out (art not created by me).

SteeringLab

Description

SteeringLab is a Unity project focused on implementing and experimenting with steering behaviors for autonomous agents. It includes dedicated scenes for missiles, flocking, and intermittent movement (jellyfish), allowing you to test, compare, and study different navigation and AI behaviors.

Video

What I did

  • Mechanics: Pursuit, Flocking, Flee, obstacle avoidance.
  • UI: Built a simple UI for controlling/testing behaviors.
  • In progress: Jellyfish steering behavior is not implemented yet.
